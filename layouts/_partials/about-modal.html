LLM disclosure

Parts of this website (HTML, Javascript and CSS) have been built with assistance from an LLM (Claude Code's Opus 4.6 model)

To be clear, all of the data present on this site has been manually curated by hand and none of it been touched by an LLM in any way.

All user-facing text (including this!) has also been written by hand with zero LLM assistance.

Most of the Javascript ends up great on the first try but a bunch of time is spent iterating on design, thinking through UX decisions and doing manual edits.

It's for these reasons that I intentionally use the term assistance rather than generated.

There is still a lot of human effort required to validate the data and get it presented in the form available here.
